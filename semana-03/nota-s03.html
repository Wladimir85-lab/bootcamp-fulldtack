1. Flexbox

Flexbox organiza elementos en una sola dirección: fila o columna. Es ideal para barras de navegación, botones y componentes internos.

La propiedad clave es display: flex en el contenedor padre. Los hijos se acomodan automáticamente.

Propiedades principales:

justify-content — alinea horizontalmente · align-items — alinea verticalmente · flex-wrap — permite que los elementos salten de línea.

2. CSS Grid

Grid organiza elementos en filas y columnas al mismo tiempo. Es ideal para el layout general de una página completa.

La propiedad clave es display: grid. Con grid-template-columns defines cuántas columnas y qué tamaño tienen.

1fr significa "una fracción del espacio disponible". grid-template-columns: 2fr 1fr 1fr crea 3 columnas donde la primera es el doble de ancha.

Para que un elemento ocupe varias columnas se usa grid-column: 1 / 4 — desde la línea 1 hasta la línea 4 del grid.

3. Responsive Design

Responsive Design significa que el layout se adapta automáticamente al tamaño de la pantalla — sin JavaScript, solo CSS.

La herramienta principal son los @media queries. Le dicen al navegador: "si la pantalla mide menos de X píxeles, aplica estas reglas".

El enfoque mobile-first diseña primero para móvil y luego agrega complejidad para pantallas grandes. Es el estándar de la industria hoy.

Ejemplo real:

@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}

Con menos de 768px → 1 columna (móvil). Con más → 3 columnas (desktop).

Resumen S03

Flexbox → una dirección, componentes internos.

Grid → dos direcciones, layout completo de página.

@media queries → el layout cambia según el tamaño de pantalla.
